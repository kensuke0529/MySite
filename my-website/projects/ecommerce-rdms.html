Project Overview

An end-to-end data engineering and AI-powered analytics platform for an eBay-like marketplace. This system features natural language SQL generation, intelligent customer service chatbots, and comprehensive business intelligence capabilities.

Key Features

AI-Powered SQL Generator:

  • Converts natural language to SQL using OpenAI GPT-4o-mini.
  • Intelligent classification to distinguish queries from conversation.
  • Automatic error handling and retry logic with LangGraph.

    Architecture

    Database: Normalized schema handling Customers, Sellers, Products, Orders, Payments, and Logistics.

    System Architecture: The platform integrates a PostgreSQL database with a Python-based runner and an AI Agent. The AI Agent generates SQL queries, which are executed by the runner, and the results are analyzed and returned as reports.

    System: LangGraph-based state machine for the SQL agent, ensuring robust error handling and multi-step query validation.

    Customer Service Chatbot

    The chatbot uses a multi-tool architecture to handle various customer requests, from policy questions to order lookups.

    Chatbot Logic Flow: The AI Assistant processes user requests and decides whether to call tools (like Order History or Policy Search). It executes the necessary tools and generates a final response based on the tool results.
