Fraud Detection Engine

Python XGBoost Scikit-Learn
Project Overview

Credit card fraud is a high-stakes problem where false negatives cost millions and false positives frustrate customers. This project focused on building a comprehensive machine learning model comparison to identify the optimal approach for detecting fraudulent transactions in highly imbalanced datasets.

Technical Implementation

I implemented and compared multiple algorithms, handling data imbalance using SMOTE (Synthetic Minority Over-sampling Technique). The final model was optimized for high precision to minimize false alarms while maintaining strong recall.

  • Logistic Regression: Baseline model for performance comparison.
  • Random Forest: Ensemble method for robust classification.
  • XGBoost (Best Performer): Gradient boosting for high accuracy and precision.
  • Isolation Forest: Anomaly detection for identifying outliers.
  • PyTorch: Deep learning approach for complex pattern recognition.
Results & Impact

The project successfully identified the optimal model for fraud detection, achieving high precision and recall while minimizing false positives.

  • 0.89 F1-Score: Achieved by the tuned XGBoost model.
  • 98% Precision: Ensured minimal false positives in transaction flagging.
  • Real-time capable: Feature engineering pipeline designed for low-latency inference.
