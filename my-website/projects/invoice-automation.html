Invoice Automation System

Automation (n8n) AI Agent Google Workspace
Project Overview

An automated invoice-processing pipeline leveraging n8n, OpenAI, and Google Workspace. The system ingests invoices from emails, extracts structured data using LLMs, detects duplicates, routes high-value invoices for human approval, stores line-item details in Google Sheets, and generates weekly financial summaries via Slack.

View Code on GitHub
The Transformation

Before automation, invoice processing was a tedious manual task prone to errors. This system transforms the workflow into a streamlined, automated process.

Manual Processing vs. Automated Pipeline
Core Features
  • Smart Ingestion: Check Gmail every minute for new invoice attachments.
  • AI Extraction: Use GPT-4o-mini to extract vendor, date, amount, and line items with high accuracy.
  • Duplicate Detection: Prevent double payments by checking invoice numbers against the database.
  • Approval Workflow: Route invoices over $10,000 for manual Slack-based approval.
  • Structured Storage: Save every line item to Google Sheets for granular analysis.
  • Weekly Insights: Auto-generate summaries of weekly spending and trends.
  • Chatbot Interface: Query invoice data using natural language.
